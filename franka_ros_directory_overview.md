# Franka ROS Repository Directory Overview

- `cmake`: CMake helper modules/macros shared across packages.
- `docs`: Documentation sources (build instructions, package docs, etc.).
- `franka_control`: ROS control integration (controllers, control loop, interfaces).
- `franka_description`: URDF/Xacro + meshes for the Franka robot model.
- `franka_example_controllers`: Sample ROS controllers to demonstrate usage.
- `franka_gazebo`: Gazebo simulation integration for Franka robots.
- `franka_gripper`: Gripper drivers and ROS interfaces.
- `franka_hw`: Hardware interface layer to the real robot.
- `franka_msgs`: ROS message/service definitions for Franka-specific APIs.
- `franka_ros`: Meta-package / top-level package dependencies and build wiring.
- `franka_visualization`: RViz configs and visualization tools.
